# HydroTune-AI 项目简介

## 项目概述

HydroTune-AI 是一个**水文模型智能率定系统**，支持多模型集成、多场次洪水率定、数据预分析、参数优化与智能报告。

## 核心功能

| 功能 | 说明 |
|------|------|
| **多模型支持** | Tank水箱模型、HBV模型、新安江模型 |
| **数据预分析** | 数据质量评估、洪水事件识别、频率分析 |
| **智能率定** | 5种优化算法（DE/GA/PSO/SCE-UA/两阶段） |
| **多场次率定** | 自动识别洪水场次，批量处理 |
| **参数导入** | 支持CSV文件直接导入参数 |
| **智能报告** | LLM生成率定报告 |

## 工作流程

```
上传数据 → 列名映射 → 时间尺度检测 → 数据预分析 → 模型率定 → 智能报告
```

## 技术架构

```
src/
├── models/          # 模型接口层（适配器模式）
├── hydro/           # 水文计算核心
├── optimizers/      # 优化算法（DE/GA/PSO/SCE/两阶段）
├── data/            # 数据处理
├── data_preanalysis.py    # 数据预分析
├── hydro_calc.py          # 水文计算与率定
├── llm_reporter.py        # LLM报告生成
└── bma_ensemble.py        # BMA集合预报
```

## 运行方式

```bash
streamlit run app.py
```

## 支持的模型

| 模型 | 参数数 | 适用场景 |
|------|--------|----------|
| Tank | 6 | 通用水文模拟 |
| HBV | 10 | 日尺度水文 |
| 新安江 | 21 | 中国湿润地区 |
