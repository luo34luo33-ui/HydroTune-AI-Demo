# XGBoost误差校正功能说明

## 一、概述

XGBoost误差校正是对水文模型模拟结果进行后处理校正的功能，通过机器学习方法预测模型误差并修正模拟结果，提升模拟精度。

## 二、使用方法

### 1. 启动应用

```bash
streamlit run app.py
```

### 2. 选择上传模式

在侧边栏选择 **"多文件（每文件一场洪水）"** 上传模式。

### 3. 配置XGB参数

在侧边栏的 **"XGBoost误差校正"** 部分：
- 勾选 **"启用XGBoost误差校正"** (默认开启)
- 设置参数：
  - `max_depth`: 树深度 (3-15，默认10)
  - `learning_rate`: 学习率 (0.01-0.5，默认0.05)
  - `n_estimators`: 树数量 (50-500，默认300)
  - `随机种子`: seed值 (1-9999，默认2025)

### 4. 开始分析

点击 **"🚀 开始分析"** 按钮，系统将：
1. 自动划分训练/测试场次（75%:25%）
2. 率定各水文模型参数
3. 自动进行XGB误差校正

### 5. 查看结果

率定完成后，在以下位置查看XGB校正结果：

1. **过程线图**：验证场次的流量过程对比图中包含XGB校正曲线（橙色虚线）
2. **指标汇总表**：包含各场次的XGB校正后NSE/KGE/RMSE/PBIAS
3. **模型表现对比**：新增XGB校正平均指标列

## 三、技术细节

### 3.1 特征工程

输入特征（共8个）：

| 特征 | 描述 |
|------|------|
| e(t-1) ~ e(t-5) | 前5个时间步的真实误差 |
| p(t-1) ~ p(t-3) | 前3个时间步的降水量 |

**注意**：校正时使用**真实误差**（flow - simulated）构建特征，而非XGB预测的误差递归。

### 3.2 训练集划分

- **训练集**：率定场次（75%）
- **测试集**：验证场次（25%）
- **固定随机种子**：seed=2025，保证可复现
- **按场次划分**：杜绝数据泄露

### 3.3 XGB模型参数

```python
XGB_PARAMS = {
    'objective': 'reg:squarederror',
    'max_depth': 10,
    'learning_rate': 0.05,
    'n_estimators': 300,
}
```

### 3.4 校正公式

```
error(t) = flow(t) - simulated(t)
corrected(t) = simulated(t) - predicted_error(t)
```

其中 predicted_error(t) 由XGB模型预测，特征为 [e(t-1), ..., e(t-5), p(t-1), ..., p(t-3)]

## 五、注意事项

1. **数据量要求**：至少需要3个率定场次才能进行XGB误差校正
2. **预热期处理**：预热期数据会被剔除，不参与训练和评估
3. **上游流量**：如果启用上游汇流，会在模型运行后叠加
4. **NSE计算**：预热期数据不参与NSE计算

## 六、模型命名

XGB校正后的模型命名格式：`{原始模型名}-XGB`

例如：
- `新安江模型2-XGB`
- `HBV模型-XGB`
- `tank水箱模型-XGB`
